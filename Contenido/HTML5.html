Etiquetas HTML5

Etiquetas semánticas:

aside: Define contenido que está aparte del contenido principal, como barras laterales.
footer: Define el pie de página de un documento o una sección.
header: Define la cabecera de un documento o una sección.
article: Define un contenido independiente y autónomo.
main: Define el contenido principal de un documento.
div: Define una división o contenedor genérico.
section: Define una sección en un documento.
nav: Define un bloque de navegación.

Etiquetas de Texto

h1-h6: Encabezados de diferentes niveles (h1 es el más importante, h6 el menos).
strong: Define texto importante (normalmente en negrita).
em: Define texto enfatizado (normalmente en cursiva).
blockquote: Define una cita larga de otro autor.
span: Define un fragmento de texto en línea.
abbr: Define una abreviatura o acrónimo.
cite: Define una referencia a una obra.
code: Define código de computadora.
q: Define una cita corta en línea.
hr: Define una regla horizontal.
br: Define un salto de línea.
p: Define un párrafo.

Etiquetas de agrupación

ul: Define una lista no ordenada (con viñetas).
ol: Define una lista ordenada (con números).
li: Define un elemento de lista.

Etiquetas multimedia

audio: Define un archivo de audio o una transmisión de audio.
source: Define los recursos de audio para el elemento audio.
video: Define un archivo de video o una transmisión de video.
track: Define texto de pista para el elemento video.

Etiquetas de tablas

table: Define una tabla.
tr: Define una fila dentro de una tabla.
td: Define una celda dentro de una fila.
th: Define una celda de encabezado dentro de una fila.
thead: Define la cabecera de una tabla.
tbody: Define el cuerpo de una tabla.
tfoot: Define el pie de una tabla.

Etiquetas de formularios

input: Define un campo de entrada para diferentes tipos de datos (texto, números, etc.).
datalist: Define una lista de opciones predefinidas para los controles de entrada.
textarea: Define un área de texto para entrada de texto más extensa.
button: Define un botón para enviar datos o realizar una acción.
select: Define una lista desplegable para elegir una opción.
option: Define una opción dentro de una lista desplegable.
label: Define una etiqueta para un campo de entrada.
fieldset: Define un grupo de campos relacionados.
legend: Define un título para un fieldset.
output: Define el resultado de un cálculo.
form: Define un formulario.
