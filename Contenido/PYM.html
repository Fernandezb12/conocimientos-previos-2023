CSS 3

Display


  • block: El elemento se muestra como un bloque rectangular.
  • inline: El elemento se muestra en línea con otros elementos.
  • inline-block: El elemento se muestra como un bloque rectangular, pero se comporta como un elemento en línea.
  • none: El elemento no se muestra.

Responsive Design “Media Queries”


  • Permite que las páginas web se adapten a diferentes dispositivos y tamaños de pantalla.
  • Se utilizan para modificar el diseño de una página web en función de la anchura del viewport.
