    • Conceptos Básicos de Git

    Git clean

    Este comando se utiliza para eliminar archivos no rastreados del directorio de trabajo. Puede ser útil para limpiar archivos generados automáticamente o archivos temporales que no deseas mantener en tu repositorio.


    Git merge

    Este comando fusiona una rama específica en tu rama actual. Se utiliza para combinar los cambios realizados en una rama con los cambios de otra rama, creando un nuevo commit que incluye los cambios de ambas ramas.


    Git log

    Este comando muestra el historial de commits del repositorio, incluyendo información detallada sobre cada commit, como el autor, la fecha y el mensaje del commit.


    Git remote add

    Este comando se utiliza para añadir un nuevo repositorio remoto. Se utiliza el parámetro add para añadir un nuevo repositorio remoto con un nombre y una URL específicos.


    Git config --global user.name

    Este comando se utiliza para configurar el nombre de usuario global de Git en tu máquina. Se utiliza el parámetro --global para que la configuración se aplique a todos los repositorios en tu sistema.


    Git config --global user.email

    Este comando se utiliza para configurar la dirección de correo electrónico global de Git en tu máquina.
