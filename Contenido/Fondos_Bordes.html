Hojas de estilo en cascada CSS 3

Fondos y bordes


  • background-color: Define el color de fondo de un elemento.
  • background-image: Define una imagen de fondo para un elemento.
  • background-repeat: Define cómo se repite la imagen de fondo.
  • background-position: Define la posición de la imagen de fondo.
  • border-width: Define el ancho del borde de un elemento.
  • border-style: Define el estilo del borde de un elemento.
  • border-color: Define el color del borde de un elemento.

Modelo de cajas


  • Content-box: El tamaño del contenido del elemento.
  • Padding-box: El tamaño del contenido del elemento más el relleno.
  • Border-box: El tamaño del contenido del elemento más el relleno y el borde.

Position


  • static: El elemento se coloca en su posición normal en el flujo del documento.
  • relative: El elemento se coloca en su posición normal en el flujo del documento, pero puede ser desplazado con respecto a su posición normal.
  • absolute: El elemento se coloca en una posición específica con respecto a su elemento padre posicionado más cercano.
  • fixed: El elemento se coloca en una posición fija con respecto al viewport.
