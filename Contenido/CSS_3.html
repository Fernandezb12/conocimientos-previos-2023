Hojas de estilo en cascada CSS 3

Tipos de selectores


  • Selectores por tipo: Seleccionan elementos por su nombre de etiqueta (ej. h1, p, img).
  • Selectores por clase: Seleccionan elementos con un atributo class específico (ej. .clase).
  • Selectores por ID: Seleccionan un elemento con un atributo id específico (ej. #id).
  • Selectores descendientes: Seleccionan elementos que son descendientes de otro elemento (ej. div p).
  • Selectores adyacentes: Seleccionan elementos que son adyacentes a otro elemento (ej. h1 + p).
  • Selectores de hermanos: Seleccionan elementos que son hermanos de otro elemento (ej. h1 ~ p).
  • Pseudo-clases: Seleccionan elementos en función de su estado (ej. :hover, :focus).
  • Pseudo-elementos: Seleccionan partes específicas de un elemento (ej. ::before, ::after).

Pseudo clases


  • hover: Se aplica cuando el cursor se encuentra sobre el elemento.
  • focus: Se aplica cuando el elemento tiene el foco.
  • active: Se aplica cuando el elemento se activa (por ejemplo, se hace clic en él).
  • visited: Se aplica cuando se ha visitado un enlace.
  • first-child: Se aplica al primer hijo de un elemento.
  • last-child: Se aplica al último hijo de un elemento.
  • nth-child(n): Se aplica al n-ésimo hijo de un elemento.
  • even: Se aplica a los elementos pares.
  • odd: Se aplica a los elementos impares.

Unidades de medida relativas


  • em: Relativa al tamaño de fuente del elemento padre.
  • rem: Relativa al tamaño de fuente raíz.
  • vh: Porcentaje de la altura del viewport.
  • vw: Porcentaje de la anchura del viewport.
  • vmin: El valor más pequeño de vh y vw.
  • vmax: El valor más grande de vh y vw.
