Tecnologías web esenciales

  1. HTML5

    Es el lenguaje de marcado estándar para crear la estructura de una página web. Define cómo se organizan los elementos en la página, como títulos, párrafos, imágenes, videos, etc.

  2. CSS3

    Es el lenguaje de estilo que se utiliza para dar formato a las páginas web. Permite controlar la apariencia de los elementos HTML, como colores, tipografías, tamaños, diseños y animaciones.

  3. JavaScript

    Es un lenguaje de programación que permite añadir interactividad a las páginas web. Se utiliza para crear animaciones, juegos, aplicaciones web y mucho más.

  4. Git

    Es un sistema de control de versiones que permite realizar un seguimiento de los cambios realizados en un proyecto. Se utiliza para colaborar con otros desarrolladores, compartir código y revertir cambios si es necesario.
